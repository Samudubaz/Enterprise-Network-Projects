Up the Interfaces
=============================
C-02(config)#interface GigabitEthernet0/0/0
C-02(config-if)#description SW-03
C-02(config-if)#ip add 172.16.1.254 255.255.255.0
C-02(config-if)#ip helper-address 172.16.0.3
C-02(config-if)#no shutdown
C-02(config-if)#exit

C-02(config)#interface Serial0/1/0
C-02(config-if)#description C-01
C-02(config-if)#ip add 172.16.4.66 255.255.255.252
C-02(config-if)#no shutdown
C-02(config-if)#exit

C-02(config)#interface Serial0/1/1
C-02(config-if)#description CBR
C-02(config-if)#ip add 172.16.4.70 255.255.255.252
C-02(config-if)#clock rate 64000
C-02(config-if)#no shutdown
C-02(config-if)#exit

Routing
=============================
C-02(config)#router rip
C-02(config-router)#version 2
C-02(config-router)#passive-interface gigabitEthernet 0/0/0
C-02(config-router)#network 172.16.1.0
C-02(config-router)#network 172.16.4.64
C-02(config-router)#network 172.16.4.68
C-02(config-router)#no auto-summary 
C-02(config-router)#exit
